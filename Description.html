In the same way that WebKit is the backbone behind many modern browsers, BagBrother is the engine behind addons like Bagnon and Bagnonium. It does nothing on it's own.

If you are using either of those two addons, you do not need to install BagBrother separately, as it is bundled automatically with your download. This entry in CurseForge exists for users which wish to download it separately.

 

Technical Explanation

BagBrother provides 99% of the addon logic. Bagnon and Bagnonium are just diiferent window layout implementations of BagBrother. That is, BagBrother provides addon modules and object classes to implement:

  • Inventory, Bank, Guild Bank and Void Storage panels.
  • Tooltip item and currency counts.
  • Item search, sorting and filters.
  • Item slot coloring, flash find, dynamic warband deposit.
  • Character offline storage and viewing.
  • Databroker carrousel.
  • Etc...
